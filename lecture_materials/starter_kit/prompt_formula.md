# The Slide DNA Prompt Formula

## The Core Equation
A consistent slide image is not random; it is a sum of four specific data points.

```
[Aesthetic Style] + [Functional Role] + [Design Intent] + [Color Palette] = PROMPT
```

---

## 1. Aesthetic Style (The "Look")
Describes the visual texture, artistic medium, or era.
*   **Examples:** "Minimalist Tech", "Paper Doodle", "Bauhaus", "Glassmorphism", "Watercolor".
*   *Tip:* Be specific about the medium (e.g., "matte plastic", "rough paper").

## 2. Functional Role (The "Job")
What is the slide used for? This determines the layout and whitespace.
*   **Opening:** High impact, center focus, bold.
*   **Structure:** Dividers, plenty of negative space, clear hierarchy.
*   **Data:** Clean backgrounds, minimal distractions, grid-like subtle overlays.
*   **Ending:** Emotional, resonant, simple.

## 3. Design Intent (The "Vibe")
The emotional keyword.
*   **Examples:** "Trustworthy", "Energetic", "Calm", "Playful", "Corporate".

## 4. Palette (The "Colors")
Control the chaos by specifying key colors.
*   **Examples:** "Neon Blue and Black", "Pastel Pink and White", "Monochrome Grey".

---

## Example Synthesis

**Input DNA:**
*   **Style:** Retro Sci-Fi
*   **Role:** Title Slide
*   **Intent:** Mystery
*   **Palette:** Purple/Teal

**Output Prompt:**
> "presentation slide background, Retro Sci-Fi style, suitable for Title Slide, Design intent: Mystery, Key colors: Purple and Teal, 8k resolution, cinematic lighting, no text."
